# 📊 SQL Retail Sales Analysis

## Project Overview

This project analyzes retail sales data using SQL and MySQL Workbench.

## Objectives

- Analyze sales performance
- Analyze profit trends
- Identify profitable regions
- Find top-selling categories
- Generate business insights

## Tools Used

- MySQL Workbench
- SQL
- CSV Dataset

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions
- LIMIT

## Analysis Performed

1. Total Sales Analysis
2. Total Profit Analysis
3. Sales by Category
4. Profit by Region
5. Sales by Segment
6. Top States by Sales
7. Average Discount Analysis

## Key Insights

- Technology category achieved highest sales.
- West region generated strongest profits.
- Consumer segment contributed maximum revenue.
- Discounts reduced profitability in some categories.
